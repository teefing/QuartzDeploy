- Windows 键 = ⌘ Command
- Enter 键 = Return 键
- Alt 键 = ⌥ Option
- Ctrl 键 = ⌃ Control
- Shift 键 = ⇧ Shift

Backspace 键 = Delete 键


有些软件定义了meta键，在win上等于 win键，在macos上等于cmd键。
